msfjsweb
====

Web interface for [MSFJS](http://github.com/SomeoneWeird/msfjs)

Installation
------------

Via git:

> $ git clone git@github.com:SomeoneWeird/msfjsweb.git

> $ cd msfjsweb

> $ npm install  

Configuration
-------------

Edit config.js

Usage
-----

> $ node server.js
